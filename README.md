# IronLog

IronLog is a lightweight workout log app you can run locally with no setup. It helps you:

- log workouts by date
- add multiple exercises to a session
- store weight and reps for each individual set
- see your latest weight for an exercise while logging
- apply starter templates for push, pull, and leg days
- browse workouts on an in-app calendar with highlighted training days
- edit or delete saved sessions
- review recent session history and simple progress cards

## Run it

Open `index.html` in your browser.

## Data storage

The app stores workout sessions in your browser using `localStorage`, so your data stays on your machine in that browser profile.
